Sie brauchen Hilfe?
  • Legen Sie eine Kundin an – geben Sie dazu einfach Name, Adresse und Telefonnummern in die entsprechenden Felder ein.
  • Klicken Sie auf “weitere Felder anzeigen”, um zusätzliche Kundendaten einzugeben und anzusehen.
  • Um mehr als eine Telefonnummer anzugeben, tragen Sie die erste Nummer ein und klicken Sie anschließend auf den Link „Telefonnummer hinzufügen“.
  • Klicken Sie unter dem Bildsymbol auf “Ändern”, um ein Kundenbild hochzuladen oder auszutauschen.
