Sie brauchen Hilfe?
  • Sehen Sie Ihre Bestellung noch einmal an, bevor Sie sie abschließen.
  • Um Produkte oder den Status der Bestellung zu ändern, klicken Sie auf Bearbeiten.
  • Schicken Sie Ihrer Kundin eine Bestätigung per E-Mail.
  • Erstellen Sie eine Rechnung im PDF- oder Excel-Format. Sie können eine Nachricht eingeben und/oder angeben, ob es sich um ein Geschenk handelt.
  • Um die Bestellung abzuschließen, klicken Sie auf Abgeschlossen oder Archivieren.
