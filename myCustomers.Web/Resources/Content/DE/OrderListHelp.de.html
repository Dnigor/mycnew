Sie brauchen Hilfe?
  • Nutzen Sie die Funktionen Suche, Erweiterte Suche, Filtern und Sortieren, um Bestellungen schneller zu finden.
  • Wenn Sie eine Bestellung ansehen möchten, klicken Sie einfach darauf.
  • Sie können den Zahlungsstatus einer oder mehrerer Bestellungen ändern, indem Sie auf das Kästchen neben den Bestellungen klicken und eine andere Option aus dem Drop-down-Menü auswählen.
  • Sie können Bestellungen archivieren oder löschen. Klicken Sie die Kästchen neben den Bestellungen an, dann klicken Sie auf Archivieren.
  • Mit der Menüoption Seitengröße können Sie die Anzahl der Bestellungen ändern, die auf einer Seite angezeigt werden.
