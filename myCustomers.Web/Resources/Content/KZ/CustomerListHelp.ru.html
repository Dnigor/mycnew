Нужна помощь?
  • Для просмотра профиля клиента нажмите на столбец с информацией о клиенте.
  • Нажмите на соседнее от клиента поле для установки флажка:
    • Печать, создать Ярлык или Экспортировать информацию о клиенте.
    • Размещение клиента в Архиве дезактивирует его. В последующем вы сможете найти данного клиента при работе с фильтром, выбрав во всплывающем меню «Архивированные».
  • Для поиска группы клиентов по определенной категории, используйте такие функции как Поиск, Фильтр, Сортировка. Например, для быстрого поиска клиента, проживающего в вашем городе, вам необходимо ввести в строку поиска название города.
  • Для очистки данных в поисковой строке нажмите значок «Х».
  • Нажмите на ярлык Заметки, чтобы просмотреть и найти особые записи/отметки клиента. Например, если вы отмечали у нескольких клиентов профессию «Преподаватель» в разделе Отметки, то при последующем поиске данных клиентов вам будет достаточно просто ввести ключевое слово в поиск.
  • Нажмите на иконку Портрет для просмотра фотографии клиента.
