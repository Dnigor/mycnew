Sie brauchen Hilfe?
  • Um das vollständige Profil einer Kundin einzusehen, klicken Sie einfach auf die Kundin.
  • Klicken Sie das Kästchen neben der Kundin an, um
    • Listen und Etiketten zu drucken oder Kundendaten zu exportieren.
    • inaktive Kunden zu archivieren. Diese Kunden finden Sie später schnell wieder, wenn Sie im Filter auf “Archiviert” klicken.
  • Finden Sie bestimmte Kundengruppen mithilfe der Funktionen „Suchen“, „Filtern“ oder „Sortieren“. So finden Sie zum Beispiel schnell alle Kunden, die in Ihrer Stadt wohnen, wenn Sie den Namen der Stadt im Suchfeld eingeben.
  • Um die Such- oder Filterergebnisse zurückzusetzen, klicken Sie auf das blaue X.
  • Klicken Sie auf den Reiter “Notizen”, um Anmerkungen zu Kunden zu suchen und anzusehen. Wenn Sie zum Beispiel bei mehreren Kundinnen „Lehrerin“ eingetragen haben, finden Sie diese, indem Sie „Lehrerin“ in das Suchfeld eingeben.
  • Klicken Sie auf das Bildsymbol, um die Bilder Ihrer Kunden einzusehen.
