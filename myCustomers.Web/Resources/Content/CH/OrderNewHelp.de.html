Sie brauchen Hilfe?
  • Sie können Produkte einfach hinzufügen, indem Sie den Produktnamen in das Suchfeld eingeben oder Produkte aus den bisher bestellten Produkten, Produktempfehlungen oder der Wunschliste der Kundin übernehmen.
  • Geben Sie die Lieferinformationen ein. Der Empfänger ist die Person, die die Bestellung erhält, wenn es sich dabei nicht um die Kundin handelt.
  • Sehen Sie den Zahlungsstatus einer Bestellung ein.
  • Fügen Sie eine Notiz hinzu, damit Sie die Bestellung leichter nachverfolgen und managen können.
  • Richten Sie ein Datum zum Nachfassen bei der Kundin ein. Dieses erscheint automatisch auf der Seite Ihrer Erinnerungen.
  • Klicken Sie auf Speichern und Bestellung überprüfen, um auf die nächste Seite zu gelangen. Hier können Sie eine Rechnung ausstellen und die Bestellung abschließen.
  • Wenn keine Rechnung benötigt wird, klicken Sie auf Abgeschlossen auf dieser Seite, um die Bestellung abzuschließen.
