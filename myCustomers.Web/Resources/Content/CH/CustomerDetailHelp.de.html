Sie brauchen Hilfe?
  • Sehen Sie sich die persönlichen Daten inklusive Profildaten sowie vergangene Produktbestellungen, automatische Produktempfehlungen und die Produkte auf der Kundenwunschliste an.
  • Klicken Sie auf Aktivitätsübersicht, um eine Liste aller Interaktionen mit dieser Kundin zu erhalten.
  • Fügen Sie eine Bestellung hinzu, bearbeiten und archivieren Sie Kundendaten, sehen Sie sich die bevorzugte Kontaktart Ihrer Kundin an und machen Sie ein Virtual Makeover für Ihre Kunden.
  • Fügen Sie eine Notiz zu einer Kundin hinzu.
  • Richten Sie eine Erinnerung ein, um bei einer Kundin nachzufassen.
